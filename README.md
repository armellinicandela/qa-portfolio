## Proyectos

### 🛒 Saucedemo — Testing funcional de e-commerce
Sitio de pruebas simulado (https://www.saucedemo.com). Proyecto completo de diseño y documentación de casos de prueba a partir de historias de usuario, gestionado en Jira.

| Módulo | Casos de prueba | Cobertura |
|--------|----------------|-----------|
| HU_001 — Iniciar sesión | 7 | Positivos, negativos y campos vacíos |
| HU_002 — Agregar / Eliminar productos del carrito | 8 | Positivos, alternativos y sin sesión |

**Documentación:**
- Historias de usuario con criterios de aceptación
- Casos de prueba con precondiciones, pasos, resultado esperado y postcondición
- Gestión del proyecto en Jira (Épica → Historia → Subtareas)
- Casos documentados en Excel
